# Titanic-Survival-Analysis-Task-2

📌 Internship Task 2

This project was completed as part of my **Data Science Internship** at **Maincrafts Technology**.

 📖About the Project

In this project, I analyzed the Titanic dataset using Python. I learned how to clean data, handle missing values, analyze passenger survival, and create visualizations.

🛠️ Tool Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Project Tasks

* Loaded the Titanic dataset
* Explored the dataset structure
* Handled missing values
* Analyzed survival patterns
* Compared survival by gender
* Compared survival by passenger class
* Analyzed survival by age group
* Created data visualizations

 📈 Visualizations

* Bar Chart: Survival by Gender
* Bar Chart: Survival by Passenger Class
* Histogram: Passenger Age Distribution



 🎯 What I Learned

This project helped me understand data cleaning, data analysis, and data visualization using Python. It also improved my practical knowledge of working with real datasets.
